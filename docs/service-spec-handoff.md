# Part 2. 서비스 플로우

## 인증·설정
- **F1 이메일 가입**: 폼(이메일·비번·닉네임·약관) → 인증번호 확인 → 계정 생성 → JWT → 홈
- **F2 카카오 로그인**: 카카오 인증 → 콜백(이메일·닉네임) → 자체 JWT → 홈
- **F3 이메일 로그인**: 폼 → 검증 → JWT → 홈
- **F4 예산 설정**: 홈 예산 카드(기본값) → 인라인 수정 → 저장 → 러닝 트래커 갱신

## 냉장고
- **F5 OCR 재고 등록**: 영수증·바코드 촬영 → OCR → 재료 NER(품목코드) → 유통기한 파싱 → 재고 저장 → (동시) 지출 → 식비 캘린더(F17)
- **F6 수동 추가**: 폼(품목·수량·유통기한) → 품목 매칭 → 저장
- **F7 수정·삭제**: 재고 목록 → 수량 수정 / 소진 / 삭제
- **F8 임박 알림**: 유통기한 추적 → D-day 감지 → Push → "이 재료 쓰는 레시피"(F11)

## 레시피
- **F9 검색·필터**: 검색어/카테고리 → ES 조회 → 목록 → 상세
- **F10 상세**: 재료·조리법 → 보유/구매 판정(재고 대조) → 부족 재료 가격(F15) → 추가비 → 영양성분
- **F11 재고 기반 추천**: 재고 입력 → 규칙 점수(활용률·임박·저비용) → 랭킹(P1 개인화)
- **F12 레시피북 저장·조회**: 상세 → 저장 → 조회/삭제
- **F13 YouTube 추출** [P1]: URL → 사전필터 → 캐시 → Gemini 추출 → NER → ES 매핑 → 레시피북
- **F14 레시피북 공유** [P1]: 공유 링크 생성 → 타인 열람·스크랩

## 가격
- **F15 가격 매기기(폴백)**: 품목코드 → 컬리 현재가 → 영수증 실구매가 → 대표가 → 미확인

## 밀플래닝 (핵심)
- **F16 "뭐 해먹지 → 장보기"**: 홈 요청 → 추천 엔진[재고+임박+예산] → 랭킹 → 담기 → 장보기 목록(차감·합산·가격) → 예산 반영 (몇 끼 사전 커밋 불필요)
- **F17 식비 기록·추적**: 영수증 OCR(자동) + 외식(수동) → 캘린더 → 월 누적 → 예산 잔여·낭비 방지
- **F18 대화형 어시스턴트** [P1]: 질문 → 입력 가드레일 → NER 분석 → 병렬 조회(재고·예산·레시피·가격) → 컨텍스트 조립 → Gemini 생성 → 출력 가드레일 → 응답
- **F19 배치 플랜** [P1]: 규모 지정 → F16을 N끼로 확장

## 횡단
- **F20 홈 대시보드**: 예산 잔여·재고·임박·추천·성과 집계
- **F21 알림 센터**: 임박(P0) / 공유(P1) → 목록 → 딥링크
